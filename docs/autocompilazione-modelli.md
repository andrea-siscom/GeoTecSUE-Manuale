# Autocompilazione modelli

Dalla voce di menu "Le mie istanze" è possibile selezionare "Autocompilazione modelli".
Una volta acceduto, è possibile selezionare il modello (ad esempio *Modello di delega alla presentazione della pratica*) e quindi selezionare (ad esempio) un *richiedente* dall'anagrafica oppure una *pratica* in compilazione.
In base alla selezione effettuata, il sistema prepara una *form* precompilata con i dati prelevati dall'anagrafica o dalla pratica che può essere modificato.
Una volta completate le modifiche, è possibile cliccare il bottone *Salva i dati*: se tutto va a buon fine, comparirà il pulsante arancione *Genera PDF*.
Il file infine generato potrà essere utililzzato all'interno della pratica di interesse (eventualmente può anche essere firmato olografo da parte della committenza, se si tratta di un documento di procura o delega).
